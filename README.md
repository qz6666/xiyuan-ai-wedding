# AI 婚礼流程系统

一个面向婚礼筹备的本地工作台，用来集中管理筹备阶段、任务、预算、宾客、供应商和 AI 文案建议。

## 当前版本

- 筹备阶段看板：定方向、订资源、做内容、落执行
- 任务管理：按阶段筛选、新增任务、切换任务状态
- 预算总览：计划预算、已支付金额、分项进度
- 宾客进度：邀请人数、确认人数、预计桌数
- 供应商状态：签约、比价、待沟通
- AI 辅助：婚礼流程、誓词、父母致辞、请柬文案的初版建议

## 本地运行

```bash
npm install
npm run dev
```

当前开发预览地址通常是 `http://localhost:3001/`。

## 验证

```bash
npm test
```

测试会构建页面并检查婚礼系统的核心内容是否能正常渲染。
